# Creating an API Key on the Anthropic Console

---

## Step 1 — Log in

Open the invite link you received via email. This will take you to the Anthropic Console and prompt you to create an account or sign in.

`https://console.anthropic.com`

## Step 2 — Select your Workspace

After login you land on the Console dashboard. Use the **workspace switcher** in the top-left corner of the sidebar and select the workspace named after your hackathon team.

## Step 3 — Open the API Keys page

In the left-hand sidebar, click **API Keys**.

## Step 4 — Create a new key

Click **Create Key**. Give it a descriptive name (e.g. your project or service), then confirm.

## Step 5 — Follow CLAUDE_CODE_SETUP.md
Follow CLAUDE_CODE_SETUP.md for instructions on using the key with Claude Code


Never commit keys to source control. Use environment variables or a vault.
Revoke any keys you are no longer using.
